# SecureBank Authentication System

## 📋 Project Overview

A secure, serverless authentication system built with AWS Lambda, featuring user registration, login with brute-force protection, failed attempt tracking, and real-time security alerts via SNS/SNS.


---

## 🏗️ Architecture

### AWS Services Used
- **AWS Lambda** - Serverless compute for authentication logic
- **Amazon DynamoDB** - NoSQL database for user data and attempt tracking
- **Amazon SNS** - Security alert notifications
- **Amazon API Gateway** - RESTful API endpoints
